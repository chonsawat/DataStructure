Node
    Node() : link=null
    Node(String) : ใส่ item และ link=null
    Node(String, Node) : ใส่ item และ โยง Node

    setItem(String) : แก้ item
    getItem() : เอาค่า item
    setLink(Node) : แก้ Node
    getLink() : เอาค่า Node

    curr : ตัวที่อยู่
    prev : ตัวก่อนหน้า

    searchData(String) : หาข้อมูล
    deleteNode(String) : หา และ ลบ Node
    insertNode(String insert, String newItem) : แทรก Node
    insertLastNode(String) : เพิ่มตำแหน่งสุดท้าย
